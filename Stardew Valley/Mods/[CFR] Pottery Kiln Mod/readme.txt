Pottery Kiln

author: sylvarmyst
version: 1.0.2

- Installation -

1. Download the latest versions of SMAPI, Json Assets, Custom Farming Redux, Spacecore, and PyTK 

	SMAPI - https://www.nexusmods.com/stardewvalley/mods/2400
	Json Assets - https://www.nexusmods.com/stardewvalley/mods/1720
	Custom Farming Redux - https://www.nexusmods.com/stardewvalley/mods/991
	Spacecore -
https://www.nexusmods.com/stardewvalley/mods/1348
	PyTK - https://www.nexusmods.com/stardewvalley/mods/1726

2. Extract mod folder from .zip to Stardew Valley/Mods. It may ask you to replace some files. These will be files in the json assets folder. Tell it to replace them

3. Run the game with SMAPI


- Known Issues -

When loading an item into the kiln, items in the inventory will shift one or two spaces towards the first inventory slot. No items will be lost due to this, it's just inconvenient and a bit irritating.
